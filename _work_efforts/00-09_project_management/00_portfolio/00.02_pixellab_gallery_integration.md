## Gallery Configuration
The `scripts/generate_gallery.py` script generates 6 images by default:
- Wizard (128x128)
- Knight (128x128)
- Dragon (128x128)
- Spaceship (128x128)
- Robot (128x128)
- Treasure (128x128)

Edit the `GALLERY_IMAGES` list in the script to customize prompts.

## Notes
- Images are pre-generated and stored in the repo (static site)
- Gallery shows placeholder if images haven't been generated yet
- Uses PixelLab API v2 for image generation
- Requires PIXELLAB_API_KEY environment variable
